Projets réalisés

Application web de gestion d’hôtel

IZEManager est une application web complète permettant la gestion quotidienne d’un hôtel. Elle inclut notamment : la gestion des réservations et de l’attribution des chambres, la vérification de la disponibilité des chambres, la génération automatique de factures pour les clients, la gestion de la consommation au restaurant de l’hôtel.

ASP.NET Core Blazor SQL Server

Application web de gestion de dossiers d’import/export

SNIT Backoffice est une application dédiée à la gestion des dossiers clients pour les opérations d’import et d’export. Elle permet de : suivre les dossiers en temps réel, centraliser les informations clients et documents, automatiser certains processus de gestion pour gagner en efficacité.

ASP.NET Core Blazor SQL Server

Ce projet est un mini-module de facturation RESTful développé en Java 17 avec Spring Boot 3.5.2.
